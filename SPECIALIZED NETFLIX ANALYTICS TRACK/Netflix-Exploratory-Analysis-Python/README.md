A comprehensive exploratory data analysis (EDA) conducted in a Python Jupyter Notebook, profiling the Netflix titles dataset through statistical summaries, data cleaning, and multi-chart visualization — culminating in an exported dashboard image.

Project Overview
Domain	Exploratory Data Analysis / Content Analytics
Tools Used	Python (Pandas, NumPy, Matplotlib, Seaborn), Jupyter Notebook
Dataset	Netflix_Titles.csv
Key Files	Netflix_Data_Analysis.ipynb, Netflix_Data_Analysis_Dashboard.png
Deliverable	Jupyter Notebook EDA + Exported Dashboard Image

What I Did
•	Built a complete exploratory data analysis workflow in Jupyter Notebook using Pandas and NumPy to load, inspect, and manipulate the raw Netflix dataset (8,000+ records).: EDA Workflow
•	Developed functions to identify and handle null values, standardize text column headers, remove duplicates, and partition features for downstream analysis.: Data Cleaning
•	Distribution of Movies vs. TV Shows (5,185 vs. 147), Top 10 Directors by title count, Top 10 Countries by content volume, Titles released per year (1940–2021), and Movie Duration distribution.: Key Visualizations Built
•	Automated chart compilation and export to Netflix_Data_Analysis_Dashboard.png — a single multi-panel image summarizing all key findings.: Dashboard Export

Key Findings
•	Jan Suter is the most prolific director with 20 titles; Raul Campos follows with 19.
•	USA leads content production with 1,846 titles; India is second with 875 titles.
•	Most movies on Netflix fall within the 80–120 minute duration range.
•	Netflix content production grew exponentially after 2015, peaking around 2019–2020.
